🎯 Challenge Objective: Find the hidden flag within the Discord server

💡 Key Clue: "Look everywhere on the Discord server"

😤 The Frustrating Journey

🤖 Step 1: The Discord Bot Red Herring

💔 Initial Struggles:

This challenge was absolutely hell for me! From the moment it went live, I was constantly trying to interact with the Discord bot on the CTF server using the /flag command.

Every time I requested /flag, the bot responded with completely different recipes instead of the actual flag. This was clearly designed as a red herring to frustrate participants.

Discord Bot Response Pattern: /flag → "Am vazut flag-ul pe aici... poate in aceasta reteta?
Pasta Aglio e Olio

Ingrediente:
• 200 g spaghete
• 3-4 căței usturoi feliați
• 2-3 linguri ulei de măsline
• fulgi de ardei iute
• pătrunjel
• sare
• piper
• parmezan (opțional)

Pași:
Fierbe spaghetele al dente în apă cu sare.
Călește usturoiul în ulei 30-60s, adaugă fulgii de ardei.
Adaugă pastele scurse + puțină apă din fierbere, amestecă.
Potrivește de sare/piper, adaugă pătrunjel și parmezan."

🔄 Desperate Attempts: I even tried adding the bot to my own Discord server to see if the behavior would change, but it just responded with "This command can only be used in a specific channel," even in DMs.

🔍 Step 2: Manual Server Investigation

After the clue was released stating to "look everywhere on the Discord server," I began a systematic investigation:

  • 📋 Checked every channel one by one
  • 👥 Examined administrator roles and permissions
  • 🔤 Looked for hidden patterns in role names
  • 📝 Searched channel descriptions and topics
  • 🎭 Investigated server emojis and custom content
Result: Unfortunately, no luck with any of these approaches...

💡 The Accidental Discovery

🎉 Step 3: The Sticker Eureka Moment

✨ The Breakthrough:

After hours of fruitless searching, I accidentally clicked on the button to send stickers while browsing the Discord interface.

On the CTF server's sticker panel, there was this strange sticker that immediately caught my attention. At first glance, it looked suspiciously like a QR code!

🔍 QR Code Discovery

The Hidden Sticker: Among the normal Discord stickers, there was one that appeared to be a QR code pattern

💡 Pro tip: Sometimes the most obvious hiding places are the ones we overlook!

🤖 Step 4: AI-Assisted QR Decoding

Since I couldn't read the QR code with my eyes, I decided to leverage technology:

Process: Discord Sticker → Download Image → Upload to ChatGPT

I downloaded the sticker image and asked ChatGPT if it could kindly read the contents of the QR code for me.

🎯 Success! ChatGPT successfully decoded the QR code and revealed it was a link to example.com with a URL parameter containing the flag!

QR Code Content: http://example.com/?flag=ctf{[FLAG_VALUE_HERE]}
Currently there is no flag because this QR code is disabled.

🏆 Solution Summary

🎯 The Complete Attack Chain

  1. Discord Bot Interaction: Multiple failed attempts with /flag command (red herring)
  2. Server Investigation: Systematic search through channels, roles, and server features
  3. Accidental Discovery: Stumbled upon sticker panel while browsing
  4. Pattern Recognition: Identified QR code-like pattern in server sticker
  5. Image Extraction: Downloaded sticker as image file
  6. AI-Assisted Decoding: Used ChatGPT to decode QR code contents
  7. Flag Extraction: Found flag in URL parameter of decoded link

⚠️ Key Lesson: The flag was literally hiding in plain sight as a "decorative" sticker that everyone could see but nobody thought to examine closely!

📚 Lessons Learned

  • Persistence Pays Off: Even when facing constant misdirection, keep trying different approaches
  • Think Outside the Box: The solution wasn't in obvious places like channels or bot commands
  • Examine Everything: Discord features like stickers, emojis, and custom content can hide clues
  • Leverage Available Tools: AI tools like ChatGPT can be valuable assistants for technical tasks
  • Red Herrings are Real: Challenge creators deliberately include misleading elements
  • Accidental Discoveries: Sometimes the best finds happen when you're not actively looking
  • Hidden in Plain Sight: The most effective hiding spots are often the most obvious ones
